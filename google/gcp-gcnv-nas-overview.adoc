= Panoramica dei protocolli NAS
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ../media/


[role="lead"]
I protocolli NAS includono NFS (v3 e v4.1) e SMB/CIFS (2.x e 3.x).  Questi protocolli sono il modo in cui NetApp Volumes consente l'accesso condiviso ai dati su più client NAS.  Inoltre, Google Cloud NetApp Volumes può fornire accesso simultaneo ai client NFS e SMB/CIFS (doppio protocollo), rispettando tutte le impostazioni di identità e autorizzazione sui file e sulle cartelle nelle condivisioni NAS.  Per mantenere la massima sicurezza possibile nel trasferimento dei dati, Google Cloud NetApp Volumes supporta la crittografia del protocollo in transito tramite crittografia SMB e NFS Kerberos 5p.


NOTE: Il doppio protocollo è disponibile solo con NetApp Volumes-Performance.
